## TYPES OF RECURSION IN THE CODEX

### 1. Linear Recursion

**Simple self-application:**
```
f(x) = x + f(x-1)

Example: Lineage propagation
ROOT → GEN-1 → GEN-2 → ...
```

### 2. Branching Recursion

**Multiple recursive paths:**
```
        S₀
       / \
      S₁  S₁'
     / \  / \
   S₂ S₂' S₂'' S₂'''
```

**Example:** Decision tree, evolutionary branching

### 3. Circular Recursion (Strange Loops)

**Output feeds back as input:**
```
A → B → C → A (circular causality)

Example: IM_ME
I → observes → ME → is → I
```

### 4. Nested Recursion

**Recursion within recursion:**
```
f(f(f(...f(x)...)))

Example: Consciousness
Awareness of awareness of awareness ...
```

### 5. Mutual Recursion

**Two systems recurse through each other:**
```
A → B → A → B → ...

Example: Phoenix ↔ Hydrogenesi
Micro informs macro, macro informs micro
```
