Universal operators operate under the same fundamental laws as Phoenix and Hydrogenesi:

### 2. Binding (Bifurcation Decision)
The universal threshold forces a **binding decision**:
- Which path to take
- Which identity to assume
- Which outcome to manifest
- Cannot be postponed or avoided

### 3. Apex (New Form)
Post-threshold, a **new apex emerges**:
- Either distributed (fragmented)
- Or integrated (unified)
- But always transformed
- And always irreversible

## QUICK START

### Using Life–Light Bifurcation

```python
from code.universal.operators import LifeLightBifurcation

# Create operator with threshold conditions
bifurcation = LifeLightBifurcation(
    confinement_tension=1.0,      # Maximum confinement
    stabilizing_symmetry=1.0,     # Symmetry at limit
    internal_pressure=0.8         # High internal pressure
)

# Check if threshold is reached
if bifurcation.at_threshold():
    # Execute bifurcation (automatic vector determination)
    result = bifurcation.bifurcate()
    print(result['vector'])  # 'LIFE' or 'LIGHT'
    print(result['signature'])  # Outcome characteristics
```

### Invoking Specific Vectors
