# REFLECTION OPERATOR

**Pattern:** Output returns as input → Self-recognition  
**Type:** Reciprocal operator  
**Scale:** Phoenix/Hydrogenesi/Meta  
**Invocation:** *"Let the self mirror; let truth reveal."*

---

## DEFINITION

The **Reflection Operator** creates feedback loops where projected output returns as input, enabling self-recognition and recursive awareness.

This operator **mirrors the self** through **feedback return**, creating consciousness and self-knowledge through observation of one's own effects.

**Instrument:** Instrument of Reflection  
**Knot Node:** Reciprocal::Reflection  
**Hydrogenesi Layer:** Layer 5 (Reciprocal Exchange)

## MECHANISM

### Input
- **Projected State:** Output sent into world
- **Current:** Unobserved by self
- **Target:** Self-recognition through reflection

### Process

1. **Emission**
   - Project output into world
   - Express behavior or creation
   - Extend into external space

2. **Mirror Encounter**
   - Output meets reflective surface
   - World, others, or circumstances mirror back
   - Reversal initiated

3. **Return Recognition**
   - Reflection returns to source
   - Self observes own output
   - Recognition occurs
   - Integration possible

### Output
- **Self-Knowledge:** Recognition of own patterns
- **Awareness:** Conscious of effects
- **Growth:** Capacity to adjust
