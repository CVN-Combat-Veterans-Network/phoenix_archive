## Migration Guide

### From v2.1/v2.2

**No Migration Required.**

v2.3 adds new capabilities without changing existing systems.

**To Use v2.3 Instruments:**

1. Review instrument documentation in `/v2.3/instruments/`
2. Check operator dependencies are installed
3. Import instruments from new module paths
4. Follow integration patterns in instrument docs

**Example:**
```python
# Existing v2.1/v2.2 code continues to work
from code.phoenix.operators import FirstBinding

# New v2.3 instruments can be added
from code.v2_3.instruments.phoenix import RenewalCycleInstrument
```
