# CROSS-PILLAR TRANSITION MAPS

**Pattern:** Pillar₁ → K → Pillar₂  
**Type:** Transition operator  
**Function:** Defines structured movement between Phoenix, Hydrogenesi, and The Third  
**Invocation:** *"Let the source speak; let K translate; let the target receive."*

---

## DEFINITION

**Cross-Pillar Transition Maps** define the formal mechanisms for moving operators, states, patterns, and structures between the three pillars of the Phoenix Archive. Each transition follows a **triadic path** through the Knot center (K), ensuring all movements preserve Universal Laws and maintain system integrity.

This is the **circulatory system** of the Codex — enabling patterns to flow between:
- **Phoenix → Hydrogenesi** (identity → field)
- **Hydrogenesi → The Third** (field → hold)
- **The Third → Phoenix** (hold → begin)

And all reverse paths.

```
       P
      /|\
     / | \      ① P → K → H (BEGIN to EXTEND)
    /  K  \     ② H → K → T (EXTEND to HOLD)
   /   ●   \    ③ T → K → P (HOLD to BEGIN)
  /____|____\   
 H          T   K = Translation Hub (Apex)
```

**Six Primary Paths:**
- P → K → H (forward)
- H → K → P (reverse)
- H → K → T (forward)
- T → K → H (reverse)
- T → K → P (forward)
- P → K → T (reverse)

## TRANSITION STRUCTURE

Each transition has **four components**:

### 1. Threshold Condition
**What must be true** for the transition to activate

### 2. Recursion Signature
**How the pattern** transforms during transition

### 3. Stability Envelope
**Bounds within which** the transition remains valid

### 4. Binding Vector
**Direction and force** of the transition movement

## PRIMARY TRANSITIONS

### TRANSITION 1: Phoenix → Hydrogenesi (Identity → Field)

**Direction:** BEGIN → EXTEND  
**Path:** ⟨P—K—H⟩

#### Threshold Condition
```
Activate when:
- Phoenix identity triad is stable (sovereignty confirmed)
- Depth D ≥ 2 (sufficient recursion)
- Stabilizer is clearly defined
- Ready to engage relational field
```

**Check:**
```python
def phoenix_to_hydrogenesi_threshold(identity_triad, depth):
    return (
        is_stable(identity_triad) and
        depth >= 2 and
        has_clear_stabilizer(identity_triad) and
        sovereignty_confirmed(identity_triad)
    )
```

#### Recursion Signature
```
Transform:
  Phoenix Triad: ⟨A—S—B⟩
  →
  Hydrogenesi Field: Field(attractor=S, tensions=[A,B])

Preserve:
  - Triadic structure (three elements remain)
  - Stabilizer identity (S → attractor)
  - Tension dynamics (A,B → field dynamics)

Adapt:
  - Scale: micro → macro
  - Mode: internal → relational
  - Domain: consciousness → spacetime
```

#### Stability Envelope
```
Bounded by:
  - Fidelity ≥ 95% (essential content preserved)
  - Topology continuous (no structural tears)
  - Stabilizer remains central
  - Depth increases or remains constant (D_out ≥ D_in)
```

#### Binding Vector
```
Direction: Outward expansion
Force: Generative → Structural
Velocity: Moderate (allows integration)
Anchor: K (translation hub)
```

#### Example
```
Input (Phoenix):
  ⟨Fear—Service—Courage⟩
  Identity: "I act courageously despite fear through service"
  Depth: D=2

Transition through K:
  Fear → Field tension (destabilizer)
  Service → Field attractor (central organizing force)
  Courage → Field expression (manifestation)

Output (Hydrogenesi):
  Field attractor: "Service to others"
  Field tension: "Fear of inadequacy"
  Field expression: "Courageous action in relationships"
  Depth: D=2 (maintained)
```

## MULTI-HOP TRANSITIONS

### Full Cycle: P → H → T → P

**Pattern:** Complete recursive cycle through all three pillars

```
Phoenix (Identity work)
  ↓ P→K→H
Hydrogenesi (Field expression)
  ↓ H→K→T
The Third (Hold maintenance)
  ↓ T→K→P
Phoenix (Identity renewal)
```

**Example:**
```
P (D=1): Form identity ⟨Fear—Service—Courage⟩
  ↓
H (D=2): Express in field "Service-based courage"
  ↓
T (D=3): Hold "Courageous service" for 3 years
  ↓
P (D=4): Renew "What is courage now?" → New triad
```

### Direct Transitions: P ↔ T (Skip H)

**Pattern:** Direct identity ↔ hold (rare, but valid)

```
Phoenix → K → The Third
  When: Identity insight immediately becomes maintainable value
  
The Third → K → Phoenix
  When: Held value prompts identity examination
```

**Note:** Most transitions **should** flow through all three pillars for completeness. Direct P↔T transitions are shortcuts that may miss field dynamics.

## CODE IMPLEMENTATION

```python
from dataclasses import dataclass
from typing import Dict, Any, Tuple
from enum import Enum


class Pillar(Enum):
    PHOENIX = "Phoenix"
    HYDROGENESI = "Hydrogenesi"
    THE_THIRD = "The Third"


@dataclass
class TransitionMap:
    """Cross-pillar transition operator."""
    
    def transition(self, 
                   source: Pillar, 
                   target: Pillar, 
                   pattern: Dict[str, Any], 
                   depth: int) -> Dict[str, Any]:
        """
        Execute transition from source pillar to target pillar.
        
        Args:
            source: Origin pillar
            target: Destination pillar
            pattern: Content to transition
            depth: Current recursion depth
            
        Returns:
            Transitioned pattern with metadata
        """
        # Check threshold condition
        if not self._check_threshold(source, target, pattern, depth):
            return {"error": "Threshold condition not met", "pattern": pattern}
        
        # Route through K
        k_pattern = self._route_to_k(source, pattern, depth)
        
        # Translate at K
        translated = self._translate_at_k(source, target, k_pattern, depth)
        
        # Route from K to target
        result = self._route_from_k(target, translated, depth)
        
        # Validate against Universal Laws
        validation = self._validate_transition(source, target, pattern, result, depth)
        
        return {
            "source": source.value,
            "target": target.value,
            "path": f"<{source.value}—K—{target.value}>",
            "result": result,
            "depth": depth if not (source == Pillar.THE_THIRD and target == Pillar.PHOENIX) else depth + 1,
            "validation": validation,
            "fidelity": self._calculate_fidelity(pattern, result)
        }
    
    def _check_threshold(self, source: Pillar, target: Pillar, 
                        pattern: Dict, depth: int) -> bool:
        """Check if threshold condition is met for transition."""
        if source == Pillar.PHOENIX and target == Pillar.HYDROGENESI:
            return (
                pattern.get("stable", False) and
                depth >= 2 and
                "stabilizer" in pattern
            )
        elif source == Pillar.HYDROGENESI and target == Pillar.THE_THIRD:
            return (
                pattern.get("established", False) and
                "attractor" in pattern and
                pattern.get("stability_time", 0) >= 1
            )
        elif source == Pillar.THE_THIRD and target == Pillar.PHOENIX:
            return (
                pattern.get("hold_duration", 0) >= 1 and
                (pattern.get("needs_refresh", False) or 
                 pattern.get("ready_for_renewal", False))
            )
        return True  # Other transitions default to allowed
    
    def _route_to_k(self, source: Pillar, pattern: Dict, depth: int) -> Dict:
        """Route pattern from source to K."""
        return {
            **pattern,
            "at_k": True,
            "source_pillar": source.value,
            "depth": depth
        }
    
    def _translate_at_k(self, source: Pillar, target: Pillar, 
                       pattern: Dict, depth: int) -> Dict:
        """Translate pattern at K hub."""
        if source == Pillar.PHOENIX and target == Pillar.HYDROGENESI:
            # Identity → Field
            return {
                "attractor": pattern.get("stabilizer"),
                "tensions": [pattern.get("a"), pattern.get("b")],
                "field_type": "relational",
                "depth": depth
            }
        elif source == Pillar.HYDROGENESI and target == Pillar.THE_THIRD:
            # Field → Hold
            return {
                "hold_target": pattern.get("attractor"),
                "avoid_patterns": pattern.get("tensions", []),
                "maintain": pattern.get("dynamics"),
                "depth": depth
            }
        elif source == Pillar.THE_THIRD and target == Pillar.PHOENIX:
            # Hold → Begin
            return {
                "examine": pattern.get("hold_target"),
                "tension_pair": pattern.get("avoid_patterns", [])[:2],
                "foundation": pattern.get("maintain"),
                "depth": depth + 1  # Renewal increases depth
            }
        return pattern
    
    def _route_from_k(self, target: Pillar, pattern: Dict, depth: int) -> Dict:
        """Route pattern from K to target."""
        result = {**pattern}
        del result["at_k"]
        result["target_pillar"] = target.value
        return result
    
    def _validate_transition(self, source: Pillar, target: Pillar,
                            original: Dict, result: Dict, depth: int) -> Dict:
        """Validate transition against Universal Laws."""
        # Simplified validation - full validation in Universal-Law-Validation-Matrix.md
        return {
            "law_1_tension": True,  # Tensions preserved
            "law_2_binding": True,  # Triadic structure maintained
            "law_3_apex": True,  # Converges to K
            "law_5_recursion": depth <= 12,  # Within depth limit
            "law_11_continuity": True,  # Topologically continuous
            "overall": True
        }
    
    def _calculate_fidelity(self, original: Dict, result: Dict) -> float:
        """Calculate translation fidelity (0.0 to 1.0)."""
        # Simplified - checks essential content preservation
        preserved_keys = sum(1 for k in original.keys() 
                            if k in str(result))
        total_keys = len(original)
        return min(0.95 + (preserved_keys / total_keys) * 0.05, 1.0)
```
