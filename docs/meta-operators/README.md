**Version:** 2.4.0-alpha

## Overview

**Meta-operators** are operators that orchestrate other operators. They sit above the standard operator layer (Phoenix, Hydrogenesi, TheThird) and coordinate complex multi-operator transformations.

Meta-operators are the **conductors** of the Phoenix Archive — they don't perform transformations directly, but coordinate when, where, and how other operators execute.

---

## Usage Patterns

### Orchestration Pattern
```python
# Use META_SYNTH to coordinate multiple operators
result = META_SYNTH.orchestrate([
    FirstBinding(a, b),
    IM_ME(identity),
    PhoenixIgnition(pattern)
])
```

### Flow Routing Pattern
```python
# Use META_FLOW to route across pillars
result = META_FLOW.route(
    source="Phoenix",
    target="Hydrogenesi",
    transform=AGNReplication
)
```

### Safe Recursion Pattern
```python
# Use META_RECURSE to enforce safe recursion
result = META_RECURSE.safe_recurse(
    operator=LineageLogic,
    max_depth=50,
    stability_threshold=0.95
)
```
