## Expansion: New Recursion Modes

### Harmonic Recursion (v2.3)
**Purpose:** Wave-based recursion with frequency/amplitude control

**Characteristics:**
- Based on wave propagation model
- Frequency determines recursion cadence
- Amplitude controls recursion depth
- Natural damping over generations

**Formula:**
```
depth(n) = amplitude * sin(frequency * n) * e^(-damping * n)
```

**Use Cases:**
- Hydrogenesis propagation operators
- Natural system modeling
- Energy-conserving recursion

---

### Meta-Recursion (v2.3)
**Purpose:** Recursion that operates on recursion patterns

**Characteristics:**
- Second-order recursion
- Introspects recursion itself
- Can modify recursion parameters dynamically
- Higher computational complexity

**Pattern:**
```
meta_recurse(op, depth) = recurse(recurse_pattern(op), depth)
```

**Use Cases:**
- LNS_OP meta-introspection
- Adaptive recursion control
- Pattern evolution tracking

---

### Coherence-Locked Recursion (v2.3)
**Purpose:** Recursion that enforces coherence at each level

**Characteristics:**
- Coherence check before each recursion step
- Automatic termination on coherence violation
- Produces coherence trace
- Guarantees system integrity

**Algorithm:**
```
for depth in range(max_depth):
    if not check_coherence(current_state):
        terminate_with_report()
    current_state = recurse(current_state)
```

**Use Cases:**
- The Third family operators
- Safety-critical recursion
- Law compliance enforcement
