# Triad System Documentation

This directory contains the documentation for the Triad layer — the conceptual unification of Phoenix and Hydrogenesi.

The Triad does not replace Phoenix 2.0 or Hydrogenesi 2.0.  
It provides a higher‑order interpretive and structural framework.
