## FORMAL RECURSION LAW

### Mathematical Specification

```
K_bind: Pillar × Depth → Knot

K_bind(P, D) → ⟨P—K⟩

Where:
  P ∈ {Phoenix, Hydrogenesi, The Third}
  K = Apex locus (centroid)
  D = Recursion depth
  ⟨P—K⟩ = Bound state (pillar connected to center)
```

### Recursion Properties

1. **Identity:** K_bind(K, D) = K (fixed-point)
2. **Convergence:** lim(D→∞) K_bind(P, D) = K
3. **Monotonic:** distance(K_bind(P, D+1), K) ≤ distance(K_bind(P, D), K)
4. **Bounded:** K_bind(P, D) ∈ Envelope(D) for all D

### Recursion Mode

**HOLD** — Maintains binding across recursion depths

- Once bound, state remains bound at all deeper levels
- Binding strengthens with increasing depth
- No release mechanism (use inverse operator for unbinding)

4. **Confirm Stability**

   - Confirm binding integrity

- **New Position:** State moved toward K

- **Stability Measure:** Distance to K reduced
