### Fixed-Point Types

**Stable Fixed-Point:**
- Small perturbations decay
- System returns to x*
- Attracting behavior

**Unstable Fixed-Point:**
- Small perturbations grow
- System moves away from x*
- Repelling behavior

**Saddle Point:**
- Stable in some directions
- Unstable in others
- Mixed dynamics

**Algebraic:**

**Computational:**

- Iterative application converges: xₙ₊₁ = T(xₙ)
- Provides constructive proof
