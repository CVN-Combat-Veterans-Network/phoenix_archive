# Meta-Operators — v2.4

v2.4 introduces meta-operators: operators that orchestrate other operators across layers, pillars, and cycles.

Meta-operator set:

- `META_SYNTH` — multi-operator synthesis
- `META_FLOW` — cross-pillar routing
- `META_RECURSE` — safe recursion envelopes
- `META_TEMPORAL` — temporal orchestration
- `META_APEX` — apex-layer transitions
- `META_SEAL` — preparation for Terminal Law sealing

Each meta-operator defines:
- purpose
- behavior
- invariants
- Knot and Hydrogenesi bindings
